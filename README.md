# playwright_testing


## Setup
1. Clone the repository:
   ```sh
   git clone <repo-url>
   cd <repo-folder>```

2. Create and activate a virtual environment:
    `python -m venv venv`
    `source venv/bin/activate`

3. Install dependencies:
    `pip install -r requirements.txt`
    `playwright install`

4. Run the test:
    `pytest test_fortnine.py`

5. To exit the virtual environment:
    `deactivate`
